# 🎰 Roulette Analyzer — محلل الروليت

إكستنشن Chrome يتتبع أرقام الروليت تلقائياً ويعرض إحصائيات مفصلة في شريط جانبي.

يدعم **Stake.com** و **bet365** (Live Dealer).

---

## المميزات

- **كشف تلقائي** للأرقام من موقع الروليت
- **إدخال يدوي** للأرقام إذا لم يعمل الكشف التلقائي
- **إحصائيات كاملة:**
  - آخر رقم ظهر (ملون)
  - توزيع الألوان (أحمر / أسود / أخضر)
  - فردي / زوجي
  - أرقام ساخنة 🔥 وباردة ❄️
  - أرقام متأخرة إحصائياً ⏳
  - سجل آخر 20 رقم
- **إدارة الديلرات** — احفظ اسم كل ديلر وشوف أرقامه المفضلة تاريخياً
- **بيانات تراكمية** — تُحفظ عبر جميع الجلسات

---

## طريقة التثبيت

1. **حمّل المشروع:**
   - اضغط **Code → Download ZIP** ثم فك الضغط
   - أو نفّذ: `git clone https://github.com/fawaazq8i-dot/roulette-analyzer.git`

2. **افتح Chrome** واكتب في الشريط: `chrome://extensions`

3. فعّل **Developer mode** (زر في الزاوية العليا اليمنى)

4. اضغط **Load unpacked** واختر مجلد المشروع

5. اضغط على أيقونة الإكستنشن في شريط Chrome لفتح الشريط الجانبي

---

## طريقة الاستخدام

| الخطوة | التفاصيل |
|--------|---------|
| 1 | افتح موقع Stake أو bet365 وادخل على لعبة روليت مباشرة |
| 2 | افتح الشريط الجانبي بالضغط على أيقونة الإكستنشن |
| 3 | اختر اسم الديلر (اضغط ＋ لإضافة ديلر جديد) |
| 4 | الأرقام تُكتشف تلقائياً — أو أدخلها يدوياً من الخانة السفلية |
| 5 | شاهد الإحصائيات تتحدث في الوقت الفعلي |
